# Scoop Bucket for Volt

[Scoop](https://scoop.sh) bucket for [Volt](https://github.com/volt-api/volt) â€” the fast, offline-first API client.

## Install

```powershell
scoop bucket add volt https://github.com/volt-api/scoop-volt
scoop install volt
```

## Update

```powershell
scoop update volt
```

## Uninstall

```powershell
scoop uninstall volt
```

## About

This bucket is automatically updated when new Volt releases are published.
